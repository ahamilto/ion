class intranet.apps.eighth.models.EighthScheduledActivity(*args, **kwargs)[source]¶

Represents the relationship between an activity and a block in which it has been scheduled.

Attributes:
  • block – the EighthBlock during which an EighthActivity has been scheduled
  • activity – the scheduled EighthActivity
  • members – the User<intranet.apps.users.models.User>`s                      who have signed up for an :class:`EighthBlock
  • comments – notes for the Eighth Office
  • sponsors – EighthSponsor`s that will override the                       :class:`EighthActivity‘s default sponsors
  • rooms – EighthRoom`s that will override the                    :class:`EighthActivity‘s default rooms
  • attendance_taken – whether the EighthSponsor for the scheduled EighthActivity has taken attendance yet
  • cancelled – whether the EighthScheduledActivity has been cancelled
